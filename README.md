## Análise Comparativa

Q1 - Estruturas de Loop
- Python foi a linguagem mais simples e direta
- Java apresentou mais verbosidade
- Haskell exige conhecimento funcional
- Swift possui boa legibilidade

Q2 - Seleção múltipla
- Ruby teve melhor legibilidade
- C é mais tradicional e explícito
- Erlang é mais repetitivo

Q3 - Sem goto
- Código ficou mais estruturado
- Uso de condição no for substitui break
- Melhor legibilidade e manutenção

Q4 - For em múltiplas linguagens
- JavaScript e C++ são bem semelhantes
- Python usa while, ficando mais simples
- Go tem sintaxe flexível
- Kotlin é moderno e limpo

Opinião 

Python e Ruby foram as linguagens mais fáceis de entender e escrever.
Java e C são mais verbosos, porém mais comuns no mercado.
Haskell e Erlang são mais difíceis para iniciantes.
